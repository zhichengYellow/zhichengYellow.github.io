Tips

一 使用浏览器开发者工具（无需安装，即学即用）
这是最直接的方法，通过控制台直接把网页的限制代码“消灭”掉：

在当前网页按下 F12（Mac 用户按 Cmd + Option + I）打开开发者工具。

切换到 Console（控制台） 标签页。

在底部的输入框中复制并粘贴以下代码，然后按 Enter（回车）：

JavaScript

Object.defineProperty(document, ‘onpaste’, {value: null});
document.onpaste = null;
window.onpaste = null;
// 针对特定输入框解除限制
document.querySelectorAll(‘textarea, input’).forEach(el => {
el.onpaste = null;
el.addEventListener(‘paste’, e => e.stopPropagation(), true);
});

现在回网页试试，通常就可以直接粘贴了。

二 控制台一行代码强制倍速（最安全、免安装）
这个方法不需要安装任何东西，直接利用浏览器内核命令去改视频的播放速度：

播放网课视频时，按下 F12（Mac 用户 Cmd + Option + I）打开开发者工具。

切换到 Console（控制台） 标签页。

复制并粘贴下面这行代码，然后按 Enter（回车）：

JavaScript
document.querySelector(‘video’).playbackRate = 2.5;
(注：末尾的 2.5 就是倍速，你可以自由改成 2.0、3.0 甚至更高。HTML5 底层最高支持 16 倍速。)
